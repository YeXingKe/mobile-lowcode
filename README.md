## 🔧 安装指南
### 环境要求
- Node.js >= 20.12.0

### 安装步骤
1. 克隆仓库：
   ```bash
   git clone https://github.com/YeXingKe/mobile-lowcode.git
   ```
2. 安装依赖：
   ```bash
   cd mobile-lowcode
   pnpm i 
   ```
